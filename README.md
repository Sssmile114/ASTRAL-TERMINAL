# ASTRAL TERMINAL

**Astral Terminal** — 一个明日方舟《孤星》风格的深空数据观测物理操作台。采集宇宙信号、解锁未知天体与星座，在 AI 助手赫尔墨斯的协同下探索深邃星际。

> 原名：深空数据观测站（Deep Space Observation Station）

---

## 功能特性

- **交互式星图** — 拖拽平移、滚轮缩放，探索 40+ 天体的深空星图
- **雷达扫描系统** — 实时雷达回波，模拟信号侦测体验
- **AI 助手赫尔墨斯** — 智能对话系统，解锁时推送科学评论与随机闲聊
- **游戏化采集** — 手动/自动信号采集，渐进式解锁恒星、系外行星、星云、黑洞、星系
- **星座解锁** — 猎户座、北斗七星、夏季大三角、黄道十二宫
- **数据存档** — 浏览器 localStorage 自动保存进度
- **观测笔记** — 内置操作员手记与系统日志
- **音乐播放器** — 内置深空主题氛围音乐
- **复古科幻 UI** — CRT 滤镜、视差星空、彗星划过、热敏打印带

## 快速开始

```bash
# 直接用浏览器打开
open index.html
```

无需构建工具，纯静态页面，任意现代浏览器即可运行。

## 技术栈

- HTML5 + CSS3（Grid / Flexbox / 自定义属性 / 动画）
- 原生 JavaScript（零框架依赖）
- Web Audio API（交互音效）
- localStorage（数据持久化）
